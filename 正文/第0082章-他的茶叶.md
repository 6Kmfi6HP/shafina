# 第82章 他的茶叶

周一早上，林叙到公司的时候办公室里还没几个人。

他经过前台的时候和值班的保安点了一下头，然后穿过走廊走到自己的工位。放下包，他没有开机，而是从包里拿出了一包茶叶——他早上出门前去便利店买的。不是他平时喝的那种，是另一种日式煎茶，包装上印着淡绿色的字。他拿起那包茶叶看了看，然后站起来走向茶水间。

茶水间的灯开着，但里面没有人。窗台上那盆绿植还在——叶片上落了一层薄薄的灰，大概是周末两天没人擦拭。他伸手抹了一下最大那片叶子的表面，灰被抹开，露出一道深綠色的光澤。

他打开公用柜的门。

上次放的那包茶叶还在，但已经快空了——纸袋被捏得很薄，只剩底部一点点碎末。他伸手摸了一下那个纸袋，感受到纸袋被反复打开过的痕迹。封口的折叠处已经有些磨损了，纸面因为多次被手指捏取而起了細小的毛邊。他看著那個紙袋——在他不在場的時候，有人打開過它。很多次。

他把旧纸袋拿出来，最后看了一眼，然后扔掉了。

他把新买的煎茶放了进去。

放好的那一刻，他站在柜门前犹豫了一下——那包煎茶的包装很新，和周围那些零散的东西放在一起，显得有些突兀。但他没有把它换掉，关上了柜门。

走回工位的路上，他遇到了同事周姐。周姐端着杯子往茶水间走，看到他打了个招呼：「早啊。」

「早。」

「周末过得怎么样？」

「还行。」他说。然后他顿了顿，补充了一句：「挺好的。」

周姐笑了一下，走进了茶水间。他继续走回自己的工位。

坐下来之后他打开电脑，屏幕亮起来，但他没有马上开始工作。他靠在椅背上，看着茶水间的方向。

他不知道她今天什么时候会来。

八点四十分，她来了。

他听到了脚步声——不是他能辨认出她的脚步声，而是他听到了有脚步声从门口方向过来，然后他抬头，刚好看到她走进来。她今天穿了一件浅灰色的针织开衫，里面是白色的衬衫，背着一个帆布包。她低着头看手机，没有注意到他在看她。她走到自己的工位，放下包，坐下来。她从包里拿出一個保溫杯放在桌上，深藍色的，他以前沒見過——大概是新買的。

他收回了视线。

一整个上午，他没有去茶水间。他不知道自己为什么不去——可能是因为他不想显得刻意。如果他在放完茶叶的当天就去茶水间很多次，她可能会觉得奇怪。他不想被她看出来任何东西。他专心工作。至少他努力让自己看起来很专心。

十一点二十分左右，他站起来，拿起自己的杯子，走向茶水间。他走得很自然——就是普通的、去倒水的样子。

茶水间里没有人。

他走到饮水机前，接了一杯热水。然后他打开公用柜——那包煎茶还在原位。他拿起那包茶叶，打开封口，给自己倒了一些。茶叶落入杯中的时候发出轻轻的沙沙声，干燥的叶片碰觸到杯底，然后被热水浸泡，开始缓慢地舒展。他蓋上茶葉袋的時候，注意到封口處被他捏出了一道新的摺痕——那條摺痕會讓下一個人打開的時候更方便。他希望下一個人會是她。

他端着杯子在窗边站了一会儿。窗外的天空灰白交加，阳光被薄薄的云层遮住了，光线柔和而均匀。他低头看着杯子里的茶汤——淡淡的黄绿色，清澈透亮。他端起杯子闻了一下，香气清雅，不浓烈，帶著一種像在清晨的草地上深呼吸時聞到的氣息。

就在这时，有人走了进来。

是她。

她拿着杯子，看到他站在窗边的时候动作顿了一下——很短的一顿，但她自己也意识到了，所以她立刻走进去，自然地接了一杯水。

他说：「早。」

她说：「早……不是已经中午了吗。」她笑了一下。

他也笑了一下：「对，已经中午了。」

两个人之间沉默了两三秒。她端着杯子没有马上走，他也没有马上走。茶水间里只有热水器的嗡嗡声和窗外偶尔传进来的车声。她看了一眼公用柜的方向——他看到她的视线落在那個櫃門上，停留了大約半秒，然後移開了。

然后她开口了：「你——」

「嗯？」

她的嘴唇動了一下，像在組織下一句話。他等著。茶水間的窗戶開著一條縫，風從縫隙裡滲進來，吹動了她耳邊的頭髮。那縷頭髮在她的臉頰旁邊晃了晃。

「没什么。」她摇了摇头，「我去吃飯了。」

她端着杯子走出了茶水间。他留在原地，看着她的背影消失在门口。門在她身後輕輕關上，門板的邊緣晃動了一下，然後靜止。茶水間又只剩下他一個人。熱水器的指示燈在牆角亮著——紅色的，穩定的，像一個持續的信號。

她刚才想说什么？

她在看到公用柜方向的时候想说什么？

那句話的開頭只有一個「你」字——後面跟著的，是什麼？是想問他「你換茶葉了？」還是更長的句子——「你——是不是又放了新的茶葉？」她看到了。他知道她看到了。而且她差一點說出來了。

但她在最後一刻把那句話收回去了——像她每次快要跨出一步的時候，又退回了安全距離。

他低头看着自己杯子里已经泡好的茶——茶汤的颜色比刚才深了一些，叶片的香气被热水完全激發出来。他端起杯子喝了一口。

是他早上放的那包煎茶。

味道很淡，但有一丝隐约的回甘，像秋末的阳光下晒干的草的味道。他把这杯茶喝完了，然后洗了杯子，把杯子放在架子上。手指離開杯子的時候，他發現杯子的位置和她中午洗好放在架子上的杯子只隔了一個杯子的距離。他看著那個距離——不到一隻手掌的寬度。他沒有移動她的杯子，也沒有移動自己的。他只是看了一眼，然後走出了茶水間。

放茶叶的时候他在想什么？

他在想——她看到了。她看到了他放的新茶叶。而且她差一点说出了什么。那未說完的話像一個沒有落地的羽毛，還飄在空中，等著誰來接住。

他把杯子放在桌上，坐了下来。电脑屏幕上是他打开了一半的文档，光标在第三行的位置静静地闪烁着。他盯着那个光标看了一会儿——光标一明一灭，一明一灭，像他在等待什么的那种节奏。

他把手放在键盘上，打下了一行字。

然后又删掉了。

他在心里把那包煎茶的名字记住了一—如果她喜欢的话，他可以再买。一直買，買到她習慣了那個味道，買到她哪天忍不住開口問他：「你為什麼一直放这种茶？」
