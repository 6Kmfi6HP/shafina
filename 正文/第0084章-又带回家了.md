# 第84章 又带回家了

回到家，她把帆布包放在玄关的地上，没有马上换鞋。

她站在那里，低着头看着那个包。包的底部被撑出了一个轻微的方形轮廓——纸盒的形状，隔着帆布面料隐隐约约地浮现着。她盯着那个轮廓看了十几秒，然后弯腰拉开拉链。

红色包装还在里面。

她拿出那盒泡面，握在手里。

纸盒还是凉的——在公司空调房里待了一整天，现在还带着办公室的温度。她握着它站在玄关，没有动。灯的开关在她左手边的墙上，她没有按下去。她在黑暗里站着，只有窗外透进来的路灯的光线，把玄关的地板照出一块长方形的灰白色光斑。

她拿着那盒泡面走进客厅，把灯打开了。

光亮的瞬间她眯了一下眼睛。然后她把那盒泡面放在餐桌上，拉开椅子坐下来，看着它。

她又带回来了。

这是第几次了？她数了一下——第一次是把它藏在抽屉里，第二次是拿出来又放回去，第三次是今天——带了，一整天，又带回来了。她像一台不断循环的机器，每天都在执行同一个程序：起床，打开抽屉，拿出泡面，放进包里，带到公司，带回家，放回抽屉。如果这是一个游戏，她一定是卡关的那一个。

她靠在椅背上，仰头看着天花板。

天花板是白色的，吸顶灯发出柔和的光。她看着灯光看了一会儿，眼睛有些酸涩，但没有移开视线。她就那样仰着头，让灯光照在脸上，觉得自己很可笑。

他送三盒泡面的时候可没有这样。

她回想起那天——他站在她工位旁边，把三盒泡面放在她桌上，说了句「泡面的测评结果，写完了今天给你发了」。她的表情，动作——什么负担都没有。就好像這件事對他來說沒有任何重量。而她呢？她只是想要回一盒同样的泡面，却像在做一件需要签署保密协议的事情。

她坐直了身体，拿起那盒泡面。

她用双手握着它，指腹按压在纸盒上。纸盒上印着那只狐狸，狐狸的眼睛眯成一条线，像是在微笑。她好羡慕那只狐狸——它什么也不用想，只需要被印在包装上，被人买走，被人泡开，被人吃掉。它的人生很简单。

她把泡面举到面前，说了一句：「你到底有什么可怕的。」

泡面没有回答她。

她叹了一口气，站起来，走到衣柜前，拉开最下面那个抽屉。毛衣还在，另一盒泡面还在。她把这盒也放了进去，放在那盒的旁边。两盒泡面并排躺在灰色的毛衣下面，像两个安静的秘密。

关上抽屉之后，她站在衣柜前面。

她今天有一个新的发现——

她在茶水间遇到他的时候，她把包放在台面上。他进来了，看到了她，也看到了她的包。他什么也没说，但她的心跳在那个瞬间加速了。

她怕的不是送不出去。

她怕的是送出去之后，他会怎么想。

如果他把那盒泡面放在桌上，什么也没说——像她今天这样，把它带回家，或者放在某个角落，那她该怎么办？如果他接受了，但没有反应，那她该怎么办？如果他的反应只是客气的「谢谢」，那她该怎么办？

她怕的不是送礼这件事。

她怕的是自己的期待。

因为她知道，她送出去的那盒泡面不只是泡面。那是一封没有写字的信。她想用这盒泡面对他说：「我记得你给我的。我也想像你一样，给你点什么。」她怕他看不懂这封信。她也怕他看懂了，但不回应。

她走进浴室，打开水龙头，洗了一把脸。

冷水打在脸上的时候，她閉上眼睛。水流顺着她的下巴滴落，在洗手台上发出清脆的声响。她关掉水龙头，抬头看镜子里的自己——脸上挂着水珠，头发沾湿了几缕，眼神有点疲惫，但还算平静。

她对着镜子说：「你明天可以再试一次。」

镜子里的人没有摇头，也没有点头。

她转身走出浴室，在睡衣外面套了一件薄外套，走到阳台上。十月的夜风凉凉的，吹在她还湿着的脸上，带着一种清凉的触感。楼下的街道很安静，偶尔有一辆车驶过，车灯划过路面，然后消失在转角。

她靠在栏杆上，看着远处高楼上星星点点的灯光。

那些窗户裡亮著暖黃色的光——每一個窗戶後面都有人在生活，在吃飯，在看電視，在說話。她不知道那些人在說什麼，就像他們也不會知道她此刻站在陽台上，手裡握著一盒沒有送出去的泡面，心裡翻來覆去地只想著一個人。

她想起了他今天看她的包的那一眼。

他看了一眼就移开了——很自然的动作，像他只是随便扫了一眼。但那一秒里，她感觉自己的血液流速改变了。那种无法控制的身体反应让她觉得陌生又熟悉——她已经不是第一次在他面前有这样的反应了。

她把额前的头发拢到耳后。

风吹过来的时候，她感觉自己的脸颊是烫的。

「明天。」她对着夜色说。

这一次她说了四个字：「明天一定送。」

然后她低头看了看自己空空的双手——

她忘了把泡面从包里拿出来。

它现在还在玄关的那个帆布包里。

她走回玄关，打开包看了一眼。那盒泡面安静地躺在包底。她笑了一下，是那种对自己无奈的笑。然后她把包链拉上，把包放在了玄关的鞋柜上，没有把泡面拿出来。

这样明天早上她就不会忘记带了。

聪明吧。

她走进卧室，关了灯。

躺下来之后她才想起来——如果明天她真的送出去了，那她包里就没有泡面了。那个在她包里待了一整天的重量会消失。她会觉得空空的嗎？

她不知道自己会不会习惯那种空空的感觉。

但至少她可以試一試。
