# 第0188章 她的优点

林叙也发现了一件事——

吴念没有看起来那么冷。

刚认识她的时候，他觉得她是那种不太好接近的人。话少，表情少，永远是一副"别来烦我"的样子。他花了很长时间才敢跟她说话，又花了更长时间才敢靠近她。

但现在他知道，那层冷只是表面。

她只是不擅长表达。但她会做。

有一天他加班到晚上八点多，办公室里只剩下他一个人。他埋头改一份方案，改着改着忘了时间，直到肚子叫了一声才意识到自己还没吃晚饭。

他打开手机想点个外卖，看到一条未读消息——她发来的，时间是两个小时前。

"吃晚饭了吗？"

他当时在忙，没有看到。

他正要回复，手机又震了一下。

又是一条她发来的消息，就在刚才。

"猜到你在忙没看到。我多买了一份，放在茶水间冰箱里了。你热一下再吃。"

他愣了一下。

他站起来，走到茶水间，打开冰箱——里面有一个保鲜盒，装着饭菜，上面贴了一张便利贴，写着："微波炉热三分钟。"

他把保鲜盒拿出来，在微波炉里转了三分半钟。打开之后，热气冒上来，是番茄炒蛋和米饭。很普通的饭菜，但他端着那碗饭站在茶水间里吃的时候，觉得这是他吃过的最好吃的一顿加班餐。

他吃完之后把保鲜盒洗干净，放在沥水架上。然后他拿起手机给她发了一条消息。

"吃完了。很好吃。"

她回了一个字："嗯。"

他盯着那个"嗯"字看了好一会儿。他心想——她就是这样的人。她不会说"我怕你饿着"，不会说"我特意给你买的"。她只会默默地做了，然后留一张便利贴告诉你微波炉热三分钟。

他后来从别的同事那里听说，她那天是下班之后特意去买了饭，又返回公司放进冰箱的。她住的地方和公司在反方向，来回要多走四十分钟。

他听了之后沉默了很久。然后他给她发了一条消息，不是"谢谢"，是"明天早上想吃什么"。

她想了几分钟，回了一个"豆浆"。

第二天早上他到公司的时候，桌上放着一杯豆浆。不是他买的——是她买的。

他发现她在学他。她在用他对待她的方式，反过来对他。

还有一次，他们周末一起去超市。他走在前面，路过饮料货架的时候随手拿了一瓶乌龙茶放进购物车里。她跟在他后面，什么也没说。

后来他们一起结账回家，他把东西从袋子里拿出来放进冰箱的时候，发现冰箱里多了一瓶乌龙茶。

他拿起那瓶乌龙茶看了看——是他上次喝过的那个牌子。

他转过头看她。她正蹲在客厅拆另一袋东西，头都没抬。

但他知道是她放的。

她没有说"我看你喜欢喝这个"，没有说"我给你多买了一瓶"。她就是看到了，记住了，然后下次去超市的时候顺手拿了。

他发现她的好，全都是这样的——藏在细节里的。

不需要说出来的好。

他开始在办公室里留意她。他发现她会在午休的时候，帮隔壁工位的同事收一下桌上的文件——因为窗户开着，文件被风吹到地上好几次了。他发现她会把用完的公共文具放回原位，摆得整整齐齐。他发现她会在下雨天出门前，在走廊里放一把公用伞。

没有人要求她做这些事。也没有人注意到她做了这些事。

但他注意到了。

他有一次在走廊里碰到她，很认真地说了一句："你是个很好的人。"

她被他突如其来的这句话说愣了，皱着眉看他，像是在判断他是不是在开玩笑。

他说完就走了，没解释。

他走出去好几步之后，回头看了一下——她还在原地站着，表情有点懵，但嘴角有一个很不明显的弧度。

他转回去，继续走自己的路。嘴角也是翘的。

后来有一次周日，他在家整理照片的时候，又翻到了去年的一些照片。他忽然发现一件事——去年的某一天，他的桌上莫名其妙地出现了一盒润喉糖。当时他感冒刚好，嗓子还有点不舒服，但没跟任何人说过。第二天桌上就多了一盒润喉糖。他当时以为是谁放错地方了，就放着没动，后来糖吃完了盒子也扔了。

但现在回想起来，那盒糖不是放错的。

是她。

他在手机上翻了翻日期，算了算时间——那是在他们还没在一起的时候，甚至可能是在他还没跟她说过几句话的时候。

她把一盒润喉糖放在他桌上，没有留任何字条，没有跟任何人提起。

他当时不知道。如果不是他今天忽然想起来，他可能永远都不会知道。

他靠在椅子上，盯着天花板看了一会儿。

然后他笑了。不是那种扬起来的笑，是那种从里面慢慢渗出来的笑。

他在心里想——你从那么早就开始对我好了。你自己可能都不知道。

他在心里想——你比你以为的要好得多。你只是不让人知道而已。
