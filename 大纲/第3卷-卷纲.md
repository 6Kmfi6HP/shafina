# 第 3 卷：拉扯 - 卷纲

> 章节: 第106-170章 | 时间: 2025年11月—2026年2月

## 卷摘要
两盒泡面之后，两人之间已经没有「他/她到底对我有没有意思」的疑问了。但「接下来该怎么办？」成了新的问题。本卷讲述他们从办公室暗号走向非工作约会的过程中，每一次靠近都伴随着同等程度的紧张和退缩。他们不是不够喜欢对方——是太喜欢了，反而什么都做不了。

## Strand 分布
- Quest: 55% - 从拉面店到茶水间窒息感的主线推进
- Fire: 35% - 双视角的紧张、误读、想要靠近又不敢靠近
- Constellation: 10% - 秋冬季节转换、新年氛围

## 约東触发
- 物件传情升级：从泡面/茶叶到更个人化的选择（他记得她喜欢的茶、她记他的口味）
- 主动边界：林叙仍然无法直接说出心意——只能用行动传递
- 拉扯节奏：靠近（约拉面）→ 后退（车站道别没说下次）→ 再靠近（茶水间相遇）
